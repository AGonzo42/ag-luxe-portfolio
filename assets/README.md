# assets

Place these files here:

- `Alison_Gonzalez_Resume_Rewritten.pdf` — required. The Download Resume buttons on the site link to this exact file name.
- `favicon.ico` (optional) — then uncomment the favicon line in `index.html`.
- `og-image.jpg` (optional, 1200x630) — then uncomment the `og:image` line in `index.html`.

Any other images used in client work sections can also live here — just reference them in `index.html` as `assets/your-file-name.jpg`.
